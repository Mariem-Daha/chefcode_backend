┌────────────────────────────────────────────────────────────────────┐
│ ⚠️  IMPORTANT NOTES                                                 │
└────────────────────────────────────────────────────────────────────┘

FREE TIER:
  • Services sleep after 15 min inactivity
  • Cold start: ~30-60 seconds
  • Database expires after 90 days
  • Files in uploads/ are ephemeral

PRODUCTION:
  • Upgrade to $7/month for always-on
  • Use specific CORS origins (not *)
  • Set up UptimeRobot for health checks
  • Use cloud storage (S3) for uploads

┌────────────────────────────────────────────────────────────────────┐
│ 🔧 TROUBLESHOOTING                                                 │
└────────────────────────────────────────────────────────────────────┘

Build fails:
  git update-index --chmod=+x build.sh
  git commit -m "Fix permissions"
  git push

Service won't start:
  Check logs in Render Dashboard → Logs

Database issues:
  Verify DATABASE_URL is set automatically
  Check database is in same region as service

CORS errors:
  Update ALLOWED_ORIGINS in environment variables
  Restart service after changes
